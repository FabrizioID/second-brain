# Second Brain

Repositorio de memoria estrategica por proyectos.

La regla central: `MASTER_IDEAS.md` solo guarda senales cross-proyecto. La memoria real vive separada por proyecto, y dentro de cada proyecto se divide como una base de datos:

- `PROJECT.md`: definicion del proyecto y variables principales.
- `threads/_index.md`: indice consultable de hilos.
- `threads/*.md`: hilos independientes, con fecha, tags, decisiones, riesgos y aprendizajes.
- `memory/*.md`: hechos, decisiones, tensiones, criterios y variables estables.
- `signals/*.md`: senales fechadas exportables.
- `analysis/*.md`: extractos o entregables textuales usados como fuente.

## Uso rapido

1. Leer `MASTER_IDEAS.md` para detectar conexiones.
2. Abrir `inteligencia/<proyecto>/threads/_index.md`.
3. Elegir maximo 4 threads relevantes por tags.
4. Leer `memory/` solo si la respuesta depende de hechos o decisiones estables.
5. Crear un thread nuevo cuando la sesion genere decisiones, fuentes o aprendizaje reutilizable.
