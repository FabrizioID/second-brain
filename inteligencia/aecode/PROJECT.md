# AECODE — Proyecto

## Descripcion
AECODE es una plataforma de formacion tecnica aplicada para el sector AEC (Architecture, Engineering, Construction). Su producto central en Fase 3 es una arquitectura pedagogica orientada a desempeno laboral, organizada en `Ruta -> Cluster -> Skill -> Capsula`.

## Fase activa
Fase 3 — La unidad central de producto es la **Skill**.

## Documentos fuente
- `Proyecto/AECODE_Fase3_Arquitectura_Skills.md` — Arquitectura base del sistema
- `Proyecto/AECODE_System_Instructions_Rutas_Clusters_Skills.md` — Instrucciones del assistant para disenar arquitecturas
- `Proyecto/AECODE_Estrategia_Marketing_Procesos.docx` — Estrategia de marketing y procesos

## Principio rector
La arquitectura de producto se centra en la Skill como unidad que crea valor, mide avance, monitorea desempeno y escala oferta.
