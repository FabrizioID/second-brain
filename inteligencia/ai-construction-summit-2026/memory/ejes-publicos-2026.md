## Criterio De Evaluacion Visual

Para cada eje visual:

1. Verificar si comunica el nombre publico visible.
2. Verificar si el copy traduce el eje a beneficio para ingenieros, arquitectos, constructores, empresas y decisores.
3. Verificar si la imagen tiene suficiente ADN AEC y no se vuelve generica de tecnologia.
4. Verificar si sponsor, respaldo y CTA no compiten con el mensaje principal.
5. Si existe diferencia entre nombre publico y capa tecnica, no corregir el nombre publico; ajustar solo profundidad, copy o visual.
