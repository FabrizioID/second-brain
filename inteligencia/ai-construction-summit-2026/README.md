# AI Construction Summit 2026

Memoria estructurada del Summit. Este proyecto debe leerse como una base de datos, no como un documento lineal.

## Lectura Recomendada

1. `PROJECT.md` para variables y proposito.
2. `threads/_index.md` para elegir hilos relevantes.
3. `memory/tensions.md` si la tarea toca riesgos, sponsors, conversion, agenda o web.
4. `memory/facts.md` y `memory/decisions.md` antes de afirmar algo como canon.
5. `analysis/*.md` solo cuando se necesite fuente textual o extracto.
